Comandos bash:

    mkdir <nome da nova pasta> -> cria uma nova pasta

Comandos git:

    git add <nome do arquivo>
        git add . -> Adiciona todos os arquivos (alterados e novos) do repositório no commit

    git commit -m -> Salva localmente as alterações do repositório inserindo uma mensagem no commit

    git config <configuração especificada> -> Acessa as configurações especificada do diretório git localmente
        git config user.name -> Apresenta o nome do usuário
        git config --global user.name "Nome do Usuário" -> Altera o nome do usuário para o valor entre aspas
        git config user.email -> Apresenta o email do usuário
        git config -- global user.email "emaildousuario@email.com" -> Altera o email do usuário para o valor entre aspas

    git log -> Apresenta o histórico de commits

    git reset -> Desfaz o git add removendo todos os arquivos do commit
        git reset <hash do commit> -> Restaura o commit indicado

    git branch -> Apresenta as branchs do projeto
        git branch -D <nome da branch> -> deleta a branch indicada sem excluir os registros dos commits

    git checkout -> Move o ponteiro HEAD do git para um commit ou branch definido atualizando os arquivos do diretório para corresponderem à versão escolhida.
        git checkout <branch definida> -> Muda o ponteiro HEAD para a branch definida
        git checkout <hash do commit definido> -> Muda o ponteiro HEAD para o commit definido
        git checkout -- <>nome do arquivo> -> Desfaz as alterações do arquivo
        git checkout -b "Nome da nova branch" -> Cria uma nova branch com o nome especificado entre aspas

    git merge <nome da branch> -> Mescla as a branch atual com a branch indicada

    git clone <endereço do repositório remoto> -> Cria uma cópia local de um repositório remoto (na nuvem)

    git push <endereço do repositório remoto> <nome da branch> -> Envia o conteúdo do repositório local para o repositório remoto na branch especificada
        git push origin <nome da branch> -> Envia o conteúdo do repositório local especificamente para o repositório remoto de origem (de onde foram originalmente clonados)
